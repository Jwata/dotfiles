# Neovim
## Dependencies
```
brew tap neovim/neovim
brew install neovim
brew install ag
# Plugin manager
curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > /tmp/installer.sh
sh /tmp/installer.sh ~/.cache/dein
```

## Python3 interface
```
pyenv install 3.5.1
pyenv global 3.5.1
pip install neovim
```
